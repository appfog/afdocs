## Other Features {#features}

### Top Priorities

* Persistent Distributed Filesystems
* Root Domains
* Custom SSL
* Group/Team Support

### Coming Soon

* Dedicated Databases (MySQL, PostgreSQL, MongoDB)
* FTP Access
* Wildcard Domains
* Scheduled Jobs (cron)
* Github Hooks
* Websockets
